# scruffy-bird
A flappy bird clone made in pygame, filled with inside jokes. It's a bit tricky to explain to someone that's new to it.

The game is very early in development and probably will never be finished. It recently got an update 4 years later.

There are probably all sorts of bugs, so please use with caution.

## Controls

Left Mouse Click
- Skip splash screen
- Interact with buttons
- Flap the scruff

Right Mouse Click
- Use power-up when charged

Space Bar
- Skip splash screen
- Flap the scruff

## Current Features

- a few more graphics
- multiple playable birds
- splash screen
- flappy bird pipe jumping action; except my scruff mate Joe is flappy bird
- menu screen
- high score tracking

## Planned Features

- scruffybirdle

## Installation

Download the project and run [main.py](main.py), provided pygame and numpy are installed

## Requirements

- python 3.14.2
- pygame-ce 2.5.7
- numpy 1.20.3
